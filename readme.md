# JoseVieira ME - Laravel Support

## Install
> Via composer
``` bash
composer require josevieirame/laravel-support
```


## Requirements

The following versions of PHP are supported by this version.

* PHP 5.6
* PHP 7.0
* HHVM

## Testing
``` bash
vendor/bin/phpunit
```
